# eda-smartphone-marketplace
# Exploratory Data Analysis – Smartphone Marketplace

## Objective
This project aims to analyze smartphone marketplace data to identify key factors influencing sales performance such as price, rating, and product visibility.

## Dataset
- Period: Jan–May 2020  
- Features: price, daily_sold, daily_view, rating, GMV
- The dataset used in this project is available in the `data/` folder.

## Key Insights

### 1. Price Distribution
- Right-skewed distribution  
- Majority in low–mid range  
- Presence of outliers  

Insight: Market dominated by mid-low segment

---

### 2. Price vs Sales
- Weak negative correlation  

Insight: Price is not the main driver of sales

---

### 3. View vs Sales
- Strong positive correlation  

Insight: Visibility significantly impacts sales

---

### 4. Rating vs Sales
- No consistent relationship  

Insight: High rating does not guarantee high sales

---

### 5. Key Driver
- Daily view is the strongest factor  

Insight: Marketing exposure is critical

---

## Tools
- Minitab / Excel

## Conclusion
Sales performance is more influenced by visibility and exposure rather than price or rating.
